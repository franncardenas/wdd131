Site Plan: Wheelchairs for All

Site Name

Wheelchairs for All - This name reflects the mission of the organization: providing free wheelchairs to those in need within the community.

Site Purpose

The website will serve as an informational hub where visitors can learn about the organization, request a wheelchair, donate, and volunteer.

Scenarios

  • How can I apply to receive a wheelchair?
  • Where can I donate or volunteer to support the organization?

Color Schema

Primary Color: #0077B6 (Strong Blue - Used for Headings)

Secondary Color: #38B000 (Vivid Green - Used for Buttons and Accents)

Typography

Headings: Arial, sans-serif

Body Text: Verdana, sans-serif
